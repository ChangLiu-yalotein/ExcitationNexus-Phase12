# Gate 2-F0 validation-similarity amendment

The frozen Gate 2-A similarity file was discovered to contain test partitions only. It was therefore not read by the corrected Gate 2-F0 analysis. Before any valid shift statistic existed, the source was replaced by a target-free validation-only recomputation from frozen acceptor component identities using Morgan radius 2, 2048 bits, chirality enabled, and nearest-train Tanimoto similarity. The admission rules are unchanged.
