# Gate 2-F0 feature firewall

All admitted candidates are PM6/DFT ground-state or DFT-S0 geometry quantities available before TDDFT. No TDDFT/Multiwfn raw file, test artifact, main Parquet, or final673 asset was read. PM6 raw energy, DFT/PM6 dipole-component deltas, IDs, split/provenance fields, and absent atom-charge/orbital fields are rejected.
