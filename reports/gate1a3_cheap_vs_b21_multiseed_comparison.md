# Gate 1-A3 cheap versus B2-1 multiseed audit

This is a descriptive post hoc audit on 1,097 already inspected common test SIDs, not a new confirmatory superiority test.

Cheap MAE: `0.070209911557` eV. Three-seed B2-1 ensemble MAE: `0.074114945713` eV. Mean paired error difference (cheap minus B2-1): `-0.003905034156` eV, bootstrap 95% CI `[-0.00852346577805659, 0.0005099794850721533]`. Negative values favor cheap.

The locked project comparison p=0.145 remains the publication wording; no statistical-superiority claim is made here.
